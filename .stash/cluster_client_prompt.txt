Build a self-contained Python CLI for a 2-slave AI cluster. Stdlib only.

# Goal
Create /home/spids/cluster/bin/clusterctl — a single Python file (executable, +x) that:
1. Parses /home/spids/cluster/cluster.toml (use tomllib — it's stdlib in 3.11+)
2. CLI subcommands: `clusterctl health`, `clusterctl ask "task text"`, `clusterctl route "task text"`
3. `clusterctl route "text"` — keyword scoring against cluster.toml [routing].code_keywords and .chat_keywords. Returns JSON: {"routed_to": "slave-a"|"slave-b"|"manual", "score_code": N, "score_chat": N, "matched_code": [...], "matched_chat": [...]}
4. `clusterctl ask "task text"` — routes the task, sends it to the chosen slave, returns raw response. Use urllib.request for HTTP (no requests lib). For Ollama slave-a: POST {base_url}/api/chat with {"model": "...", "messages": [{"role": "user", "content": "..."}], "stream": false}. For LM Studio slave-b: POST {base_url}/chat/completions with {"model": "...", "messages": [...], "stream": false}. Timeout: 120s for slave-a, 180s for slave-b. JSON output: {"slave": "slave-a"|"slave-b", "model": "...", "content": "...", "latency_s": N}
5. `clusterctl health` — pings each slave's base_url (GET / for ollama returns OK, GET /v1/models for lmstudio). Verifies the configured model is loaded (compare to /api/tags or /v1/models output). JSON output per slave: {"name": "slave-a", "ok": true|false, "latency_ms": N, "model_available": true|false, "error": "..."}
6. Log every ask() to /home/spids/cluster/logs/ask.log with timestamp, slave, model, task, latency, first 200 chars of response. Use logging module.

# Hard rules
- Single file at /home/spids/cluster/bin/clusterctl
- Shebang: #!/usr/bin/env python3
- chmod +x after creation
- All paths absolute or relative to script location
- Use tomllib (Python 3.11+) — check Python version first
- Match case-insensitive AND accent-insensitive (normalize unicode: unicodedata.normalize('NFKD', text) + strip combining marks)
- Score = number of DISTINCT keywords matched. Tie goes to slave-b (code). If both scores are 0, routed_to = "manual".
